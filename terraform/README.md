# Terraform

Este diretório concentra os arquivos de infraestrutura como código do projeto.

## Estrutura sugerida

- `main.tf` para recursos principais
- `providers.tf` para provedores
- `variables.tf` para entradas do módulo
- `outputs.tf` para saídas
- `environments/` para separação por ambiente
- `modules/` para componentes reutilizáveis
